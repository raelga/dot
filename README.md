dot
========

Bork-based dot files setup 

### Features
 
* Dot files using bork.
* zsh + antigen

### Install

1. git clone https://github.com/raelga/dot.git ~/.dot
2. cd ~/.dot
3. ./dot

### Commands
 
```
# Install and update bork and dot files.
$ ./dot
```

### Info

```
./borks
└── Bork files.
./config
└── Config files.
```
