dotfiles
========

Bork-based dotfile setup 

### Features

* Dot files using bork.
* zsh + antigen

### Install

1. git clone https://github.com/raelga/dot.git ${HOME}/.dot
2. cd ${HOME}/.dot
3. ./dot

### Commands

```
# Install and update bork and dot files.
$ ./dot
```

### Info

```
./borks
└── Bork files.
./config
└── Config files.
```
